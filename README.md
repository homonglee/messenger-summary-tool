# 메신저 내용 정리

메신저 텍스트를 붙여넣거나 텍스트 파일로 업로드하면 대화 내용을 보기 좋게 구조화하고 TXT, MD, DOCX로 저장할 수 있는 1페이지 웹 도구입니다.

## 기능

- 텍스트 붙여넣기 및 `.txt/.md/.csv/.log` 업로드
- UTF-8 / EUC-KR 텍스트 파일 읽기
- 참여자, 핵심 요약, 결정사항, 할 일, 질문, 링크 자동 분류
- 프리뷰 / Markdown / TXT 탭
- TXT, MD, DOCX 다운로드
- 브라우저 내부 처리 — 별도 서버 업로드 없음

## 실행

정적 파일이므로 `index.html`을 브라우저에서 열거나 정적 서버로 배포하면 됩니다.
